Following are the source codes for a few of my projects:

Temporal abstraction in reinforcement learning using ant system (Java)

For my Master's thesis, I worked on temporal abstraction in reinforcement learning to speed up the learning process in large environments. I proposed and implemented a new method that learns the subgoals based on pheromone changes during a few iterations of the ant system algorithm. Following is an implementation of my proposed method, together with several benchmarking environments including Grid World, Taxi, Hanoi Towers and Playroom, written in Java.

A simple method for detection of white matter hyperintensities (C++, OpenCV)

This is actually a 10 days project I worked on a few months before starting my PhD that helped me better understand the upcoming challenges of my project, as my PhD was planned to be mainly about neuro-image analysis and in particular detection/segmentation of white matter hyperintensities in the brain MR images.
The source code is implemented in C++ and uses openCV for the basic image processing functionalities and applies Adaboost with Support Vector Machines as the classification method.

Training a convolutional network with a multi-scale late-fusion architecture (Python, Theano)

This code is implemented in pure Theano (Python) for training a network that gets patches of multiple scales from an H5 dataset, shares the filters among the different streams, fuses the resulting appearance feature vectors and passes them over a few more fully connected layers. The architecture of the network and the learning parameters are taken in a separate .lpd file.
